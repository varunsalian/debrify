Export from Stremio Web

Install the browser extension, open Stremio Web, then download the addon export JSON.

1

Open the extension page

Open the Stremio Addon Importer listing in the Chrome Web Store. This extension creates the JSON file that Debrify can import.

Open Chrome Web Store

Open the Stremio Addon Importer extension listing
2

Install the extension

Click Add to Chrome, then confirm with Add extension. Chrome shows the Stremio domains the extension can access.

Add to Chrome Add extension Stremio Web access
Confirm the extension install in Chrome
3

Open Stremio Web and sign in

Open Stremio Web, sign in, then open the extension from the browser toolbar. Click Import from this tab.

Click here to open Stremio Web

Sign in first Import from this tab
Use the extension while Stremio Web is open and signed in
4

Download the JSON export

After the extension reads your addon list, it shows a preview. Click Download JSON and keep that file ready for Debrify.

Preview addons Download JSON
Download the exported addon JSON file

Import into Debrify

Use the JSON file from the extension to populate Debrify's Stremio addon list.

5

Open Addons in Debrify

In Debrify, open Addons and stay on the Stremio Addons tab. Click Import JSON, then select the file you downloaded from the extension.

Addons Stremio Addons Import JSON
Import the JSON export from Debrify's Addons screen
6

Review the import result

Debrify shows how many addons were found, imported, already installed, or unsupported. Click Done, then review the addon list.

Imported Already installed Unsupported skipped
Debrify summarizes the imported addons
In short

Install the Chrome extension, sign in to Stremio Web, use Import from this tab, download the JSON file, then import that file from Debrify's Addons screen.

Best path for existing Stremio users

This import flow is the quickest way to bring your existing Stremio addon setup into Debrify without manually pasting each manifest URL.
