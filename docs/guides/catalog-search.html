Movie Search Flow

Use catalog search when you want Debrify to find the title first, then decide how you want to open it.

1

Search for the movie

Enter the movie name in catalog search. Debrify shows matching title cards with quick actions on each result.

Sources Play More menu
Catalog results for a movie search
2

If you tap Play

Debrify automatically picks a source for the movie and starts playback. This is the fastest path when you do not want to choose a source manually.

Automatic source selection Starts playback
Use Play for the quickest route into the movie
3

If you tap Sources

Debrify opens the source list for that movie. From there, you can sort, filter, and choose the source you want.

Manual source selection for the chosen movie
4

After selecting a source

Once the source is selected and added, Debrify shows the available actions for that movie.

Open Play now Download to device Add to playlist Add to channel
Actions shown after the movie source is ready
5

Use the menu for title actions

The 3-dot menu on the movie card is for title-level actions. This is separate from playback and source selection. Trakt options appear only if your Trakt account is connected.

Trakt actions Select Source Add to Stremio TV
Menu actions for the selected movie
In short

For movies in catalog search, use Play if you want Debrify to pick a source automatically. Use Sources if you want to choose the source yourself.

Series Search Flow

Series search supports direct play, episode browsing, and optional bound sources for faster and more reliable episode playback.

What is a bound source?

A bound source is a complete series or season source that you attach to a show once. After that, Debrify can reuse it to match the correct episodes automatically instead of searching again every time you press Play.

1

Search for the series

Search for the series in catalog mode. Series results show Episodes, Play, and the menu on each result card.

Episodes Play More menu
Series results in catalog search
2

If you tap Play on the series

Debrify searches for the episode where you left off and starts playback from there. If a bound series source is already set, Debrify uses that source to pick the correct episode automatically. If no bound series source is set, Debrify searches each time and picks the episode automatically, which can be slower.

Resume where you left off Uses bound source when available Automatic episode search Can be slower without a bound source
Play on the series card uses the resume flow
3

Browse episodes

Tap Episodes to open the episode browser for the selected series. From there, you can switch seasons and open a specific episode.

Season picker Episode list Episode-level actions
Episode browser for the selected series
4

Play or search sources for an episode

Each episode has its own Play and Sources buttons. The same logic applies here: Play auto-picks a source for that episode, while Sources opens the manual source list.

Play episode directly Open episode source list
Episode-level Play and Sources buttons
5

Bind a series source for faster playback

Use the top Source button in the episode browser to manage a series-level source. A bound source is a complete series or season source that you attach to the show once, so Debrify can keep using it to match the correct episodes automatically.

Faster repeat playback Better episode matching Works across episodes
Bound sources are used before fresh search
6

Add or choose a bound source

From the series source panel, you can add a new source by searching torrents or choose one from your cloud provider. This is how you set up a bound series source.

Torrent Search Real-Debrid TorBox
Choose how to add a bound source
7

Search sources for a specific episode

If you want to choose a source manually for one episode, use Sources on that episode. Debrify opens the episode-specific source list so you can select exactly what you want.

Episode-specific search Manual source selection
Source list for a single episode
8

After selecting the source

Once the source is ready, Debrify shows the available actions. This works the same way after a series or episode source is selected.

Open Play now Download to device Add to playlist Add to channel
Actions after the selected source is ready
In short

For series, Play resumes from where you left off. A bound source means you have attached a complete series or season source to the show, so Debrify can reuse it for episode playback. Without a bound source, Debrify searches each time. Use Episodes to browse, and use Sources on an episode when you want manual control.
