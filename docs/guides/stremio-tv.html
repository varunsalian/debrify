Stremio TV Guide

Turn catalogs into live channels.

Stremio TV turns addon catalogs and imported catalogs into rotating TV-style channels. You can watch the current item, open the guide, filter channels, import your own catalogs, and tune playback behavior.

Using Stremio TV

Start from the Stremio TV tab, then watch the current item or browse the guide for the channel you want.

1

Open Stremio TV

Use the main navigation to switch into Stremio TV. This opens the channel-based view powered by your installed and imported catalogs.

Open the Stremio TV tab from the main menu
2

Browse the channel rows

Each channel row shows the current item, its metadata, and the channel timing. Use Play to start the current item or Guide to open the channel guide.

Use the top menu

The top action menu gives you the main Stremio TV controls. This is where you can shuffle the lineup, refresh channels, filter visible channels, or import new catalogs.

Importing Channels

Stremio TV can use local and remote catalogs, so you are not limited to the default addon lineup.

What import does

Importing adds local Stremio TV catalogs that behave like channels. These can come from files, links, pasted JSON, repositories, or Trakt-powered lists.

Want to build your own JSON catalog?

Open the JSON Catalog Guide →

5

Open the import menu

From the top menu, open Import to choose how you want to add new channels. Debrify supports local JSON catalogs, repository-based catalogs, and Trakt-powered imports.

From File From URL Paste JSON From Repository From Trakt
All available import paths for Stremio TV catalogs
6

Choose the right import option

Use the import type that matches where your catalog already lives. You do not need every option for every workflow.

Pick one import path Same end result: a local channel catalog

From File: import a local .json catalog from your device.

From URL: import a catalog from a direct JSON link. If the link contains a Trakt-style list export, provide a catalog name.

Paste JSON: paste raw catalog JSON directly into Debrify. This also works for raw Trakt list JSON if you enter a name.

From Repository: browse JSON catalogs from a supported repo and import one or more files.

From Trakt: import watchlists, history, trending, custom lists, or liked lists. This requires Trakt to be connected first in Settings.

Need the JSON format? Open the Catalog Structure Guide →

Settings

8

Adjust playback behavior

The settings page controls how Stremio TV behaves. Key options include random episodes, preferred quality, start position, and which debrid provider should be used for streams.

In short

Stremio TV gives you a channel-style way to watch Stremio catalogs. Use Play for the current item, Guide to browse the lineup, Filter channels to clean up what you see, and Import to add your own catalog-based channels.
